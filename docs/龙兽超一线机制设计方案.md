# 龙族·兽人 后期机制升级专案（超一线设计）

> 范围：**龙族 14 人 + 兽人 68 人**（race 含 Dragon/Beast，rar≥3，非助战）。
> 重点是**机制**而非数值：把只有后期五星才有的特色机制组（固有状态循环、状态贯通、
> 独立乘区、连击引擎、Fever 引擎、发动技能动作）按流派成组移植给龙兽，
> 队长技同步加机制行。数值仍走六系文档的倍率通道，全部 0/5 结尾。

## 一、后期机制组件库（全部来自真实后期五星行，成组移植才成立）

| 组件 | 捐赠者与行 | 机制说明 |
|---|---|---|
| A 固有爆发组 | 莉莉丝111165 能力2 L1/L2/L3 | 技能发动→授予固有计数 → 指定动作消耗 → 大伤害窗口（授予/判定/消耗**必须整组移植**，固有ID 保持一致） |
| B 贯通组 | 授予行=玛丽娜111002 能力3L1「弹射≥20→状态贯通」＋生效行=莉莉丝 能力3L1/L2「持续·贯通→攻击/弹射伤害」 | 两行配对：一行授予贯通状态、一行贯通期间吃增益 |
| C 堆叠组 | 炎龙王111183 能力2-6「强化弹射≥3(限10次)→攻击/技伤」 | 高上限暖机堆叠，可换 during kind 适配流派 |
| D 连击引擎 | 罗尔夫141159 队长L2「状态连击加成」/L3「状态连击数↓」＋水艾莉亚121003 能力5「追加连击」 | 喂连击、降连击需求，全队连击轴 |

**流派→默认组件**：技能伤害→C(技伤版)＋F；强化弹射→B＋D；直击(协力球)→D＋F(Direct版)；
Fever→E；能力伤害→F(能伤版)＋触发 CT 减半。

## 二、通用包（未点名的全部龙兽）

1. 能力侧：按流派移植 1 组默认组件（append_line_adapted，元素自动适配、觉醒门槛清零、协力位可用）。
2. 队长技 +1 机制行（同表 leader→leader 复制）：技能系=「全队技能槽充能+5%」（拉夫马诺 能力6 适配）；
   弹射系=「状态连击加成+3」；直击系=「全队独立乘区Direct+5%」；Fever系=「Fever时间延长+10%」；
   能伤系=「全队独立乘区能力伤害+5%」。
3. 数值：六系文档倍率照常（龙+10%/兽+5% 目标已计入）。
